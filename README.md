# Image Editor

A browser-based image editor built with Vite, React, TypeScript, shadcn-ui and Tailwind CSS.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Fabric.js

## Development

```sh
# Install the exact reviewed dependency graph
npm ci --ignore-scripts

# Start dev server
npm run dev

# Run every local quality gate
npm run check
```

Production configuration and the rollback-aware release procedure are
documented in [`deploy/README.md`](deploy/README.md).
