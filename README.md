# Image Editor

A browser-based image editor built with Vite, React, TypeScript, shadcn-ui and Tailwind CSS.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Fabric.js

## Development

```sh
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
